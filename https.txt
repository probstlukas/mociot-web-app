macOS
1. Run self-signed-ssl.sh
2. Use Live Server extension for VS Code
3. Create .vscode folder inside the project
4. Inside that folder create settings.json file
5. Paste the following code:
{
"liveServer.settings.https": {
"enable": true, //set it true to enable the feature.
"cert": "D:\\https\\primary.crt", //full path of the certificate
"key": "D:\\https\\private.key", //full path of the private key
"passphrase": "12345"
}
}
6. Start the Live Server and access your project using HTTPS
